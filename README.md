# Orgs Hook
React Native: utilizando e criando Hooks

# Requisitos
- [NodeJs](https://nodejs.org/en/).


# Instalação 
Instalar as dependências do projeto
```sh
npm install
```

# Rodando Aplicativo
```sh
npx react-native run-android
npx react-native run-ios
```
